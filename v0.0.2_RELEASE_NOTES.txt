   Opción: 17 (Desinstalar TODOS)
   ✓ Confirma 2 veces
   ✓ Verifica que se desinstala completamente

   Opción: 14 (Instalar TODO nuevamente)
   ✓ Comprueba que vuelve a funcionar

4. VALIDACIÓN FINAL:
   $ firefox --version  # Debe fallar si está desinstalado
   $ code --version     # Debe fallar si está desinstalado
